# Lwanga Tools

This directory contains additional tools for working with Lwanga.

## Tools

- `lwanga-fmt` - Code formatter
- `lwanga-lint` - Linter for code quality
- `lwanga-doc` - Documentation generator
- `lwanga-test` - Test runner
- `lwanga-disasm` - Disassembler for Lwanga binaries

## Building Tools

```bash
cd tools
mkdir build && cd build
cmake ..
make
```
